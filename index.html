NEXT

셀리느는 신월에서 여러 마법을 배운 실력자다.
신중하게 마법을 사용하여 골렘을 쓰러뜨리자!

A 키 : 파이어볼(F) - 전방으로 화염구를 발사한다. 쿨타임:2초 / MP:3%

Z 키 : 윈드(N) - 빠르고 날카로운 바람을 발사한다. 쿨타임:5초 / MP:6%

X 키 : 썬더(E) - 강력한 번개를 일으킨다. 번개에 맞은 대상은 공격력이 감소한다. 쿨타임:20초 / MP:35%

W 키 : 웨이브(W) - 공기 중의 수분을 모아 파도를 일으킨다. 쿨타임:10초 / MP:15%

S 키 : 힐(Buff) - 마나를 사용하여 자신의 체력을 회복한다. 회복량:10% / MP:10%

TIP : 웨이브(W)는 화 속성에게 2배의 데미지를 입힌다.
